# Real Use Log

Use this file after real course or assignment sessions. Keep entries short. The goal is to find friction, not to write a diary.

## How to log

```text
Date:
Course / task:
Workflow used:
Input type:
Useful output:
Too heavy / too generic:
Where I got stuck:
Next template or rule change:
Next tiny action:
```

## Weekly Review Prompts

- Which workflow did I actually use this week?
- Which template helped me move faster?
- Which template felt too heavy?
- Did I retrieve before asking AI to summarize?
- Did any course output become an assignment, project, or interview asset?
- What one rule should be changed before next week?
